1. 컨테이너 기술이란 무엇입니까? (100자 이내로 요약)
모듈화되고 격리된 가상의 컴퓨팅 공간 또는 환경을 만드는 기술

2. 도커란 무엇입니까? (100자 이내로 요약)
이미지 기반의 컨테이너 기술로 여러 환경에서 애플리케이션 배포를 자동화하는 것

3. 도커 파일, 도커 이미지, 도커 컨테이너의 개념은 무엇이고, 서로 어떤 관계입니까?
- 도커 파일: 도커 이미지를 생성할 때 사용되는 파일
- 도커 이미지: 컨테이너를 만드는 데 사용되는 읽기 전용 템플릿, 도커 파일을 빌드하여 이미지를 만듦.
- 도커 컨테이너: 격리된 공간에서 프로세스가 동작하는 공간, 도커 이미지를 사용하여 컨테이너를 생성해줌.
