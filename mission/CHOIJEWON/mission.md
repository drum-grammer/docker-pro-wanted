- Doker file

포함하는 독립 실행형 패키지입니다

- Docker container

도커 컨테이너는 도커 이미지의 런타임 인스턴스입니다 모든 종속성과 더불어

특징으론 가벼우며 격리된 환경이기 때문에 운영체제 커널을 공유합니다
